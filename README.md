# Interactive-Business-Dashboard-Streamlit
Interactive Business Intelligence dashboard built with Streamlit for analyzing Global Superstore sales, profit, and customer segment performance with dynamic filters.

# Interactive Business Dashboard (Streamlit)

## Objective
Interactive dashboard for analyzing sales, profit, and 
segment-wise performance of Global Superstore data.

## Dataset
- **Source:** Global Superstore Dataset
- **Download:** https://www.kaggle.com/datasets/apoorvaappz/global-super-store-dataset

## Dashboard Features
- Sidebar Filters — Region, Category, Sub-Category
- KPI Cards — Total Sales, Profit, Orders, Customers, Margin
- Sales by Region (Bar Chart)
- Profit by Category (Color coded green/red)
- Top 10 Sub-Categories by Sales
- Sales by Customer Segment (Pie Chart)
- Top 5 Customers by Sales
- Filtered Raw Data Table

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Place dataset CSV in same folder
4. Run dashboard:
   streamlit run app.py
5. Open browser: http://localhost:8501
